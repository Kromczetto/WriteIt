# User Guide

This document describes how to use the application from the perspective
of an end user.

---

## Account Management

### Registration

To create a new account:

- Navigate to the registration page.
- Provide a valid email address and password.
- After successful registration, log in to the system.

### Login

To access an existing account:

- Enter the registered email address and password.
- Upon successful authentication, the user is redirected to the profile page.

---

## Article Store

The article store allows users to browse and discover published content:

- Browse available articles.
- Search for articles using keywords.
- View article previews and user ratings.

---

## Renting Articles

Users can rent articles for a selected duration:

- Choose a rental period:
  - **1 day**
  - **7 days**
  - **30 days**
  - **Unlimited access**
- Confirm the rental.
- Rented articles become available in the user library.

---

## Reading Articles

Rented articles can be accessed as follows:

- Open articles from the library or directly from the store.
- Read the full article content.
- Download the article as a **PDF** file for offline access.

---

## Profile Management

The user profile provides tools for content and account management:

- View personal account information.
- Create new articles.
- Edit or delete existing articles.
- Manage publication status (draft or published).

---

## Social Features

### Friends

The application supports basic social interactions:

- Send friend requests using email addresses.
- Accept or reject incoming friend requests.
- View and manage the friends list.

### Chat

Users can communicate with friends through real-time chat:

- Open private chat sessions with friends.
- Send text messages.
- Share rented articles directly via chat.
