# User Guide

This document describes how to use the application from the perspective of an end user.

---

## Account Management

### Registration
- Navigate to the registration page.
- Provide a valid email address and password.
- After successful registration, log in to the system.

### Login
- Enter registered email and password.
- Upon successful authentication, the user is redirected to the profile page.

---

## Article Store

- Browse published articles.
- Search articles using keywords.
- View article previews and ratings.

---

## Renting Articles

- Select a rental duration:
  - 1 day
  - 7 days
  - 30 days
  - Unlimited access
- Confirm rental.
- Rented articles become available in the user library.

---

## Reading Articles

- Open rented articles from the library or store.
- Read full article content.
- Download the article as a PDF file.

---

---

## Profile Management

- View personal information.
- Create new articles.
- Edit or delete existing articles.
- Manage publication status (draft/published).

---

## Social Features

### Friends
- Send friend requests by email.
- Accept or reject incoming requests.
- View friends list.

### Chat
- Open real-time chat with friends.
- Send text messages.
- Share rented articles via chat.
